# bbedit_odin_language_module
A Codeless Language Module and colour scheme for Odin in BBEdit on MacOs

Put Odin.plist in /Users/username/Library/Application Support/BBEdit/Language Modules
Put Odin.bbcolors in /Users/username/Library/Application Support/BBEdit/Color Schemes
Restart BBEdit if it's running.
Set up the .odin extention binding in Settings->Languages
